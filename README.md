# 驾考在线题库练习与考试管理系统
基于 Vue + Spring MVC + MyBatis + MySQL 构建的前后端分离系统，实现「学习 - 考试 - 数据管理」全流程闭环，支持管理员/普通用户差异化权限管控。

## 项目核心需求
- **登录与账号管理**：支持邮箱/手机号+密码登录，普通用户可注册/找回密码，管理员账号由超级管理员手动添加。
- **管理员功能**：
  - 管理员/普通用户的增删改查、多条件查询、Excel 批量导入导出；
  - 题库查询/查看、模拟试卷生成（标准卷/专项卷）、题库同步（对接聚合数据API）。
- **普通用户功能**：
  - 题库练习（自由练习、模拟考试、专项训练）、考试计时/自动评分/错题保存；
  - 个人信息修改、密码重置、考试记录查询、错题回溯（含解析）。

## 系统设计
### 1. 软件体系结构（分层架构）
| 层级         | 技术栈/核心职责                                                                 |
|--------------|--------------------------------------------------------------------------------|
| 视图层       | Vue 3 + Element Plus，组件化开发，Axios 实现前后端交互                          |
| 控制层       | Spring MVC，定义REST接口、参数校验、跨域配置，统一返回JSON格式结果               |
| 业务层       | Spring IoC 管理实例，@Transactional 事务控制，封装用户/题库/考试核心业务逻辑     |
| 数据访问层   | MyBatis 3，Mapper接口+XML映射文件，实现数据CRUD、多条件查询                     |
| 数据存储层   | MySQL 8.0，合理表结构设计+索引优化，存储用户/题库/考试记录等全量业务数据         |

### 2. 核心功能模块
| 模块分类       | 核心能力                                                                 |
|----------------|--------------------------------------------------------------------------|
| 登录/注册/找回密码 | 多方式登录、身份校验、JWT令牌生成、验证码发送/验证、密码加密存储           |
| 管理员管理     | 管理员账号增删改查、密码重置、Excel导入导出                              |
| 普通用户管理   | 用户全量管控、数据合规校验、Excel批量操作                                |
| 题库管理       | 题库同步、题目CRUD、多条件筛选、模拟试卷生成（标准/专项）                 |
| 驾考练习       | 模拟考/专项训练/随机测试、倒计时、自动判分、错题自动保存                  |
| 个人中心       | 信息修改、密码重置、考试记录查询、错题回溯（题干/作答/解析）              |

### 3. 数据库核心表结构
| 表名         | 核心用途                 | 关键字段                                                                 |
|--------------|--------------------------|--------------------------------------------------------------------------|
| user         | 普通用户信息存储         | user_id、username、password(BCrypt加密)、email、phone、user_type         |
| admin        | 管理员信息存储           | admin_id、username、password、email、phone、role_id、user_type           |
| question     | 题库数据存储             | id、api_id、question、answer、item1-item4、explains、subject、model、question_type |
| exam_record  | 考试记录存储             | id、user_id、subject、model、score、is_pass、exam_time                   |
| error_question | 错题记录存储           | id、user_id、api_id、user_answer、correct_answer、exam_record_id         |
| verify_code  | 验证码存储               | id、email、code、expire_time、used                                       |

## 项目总结
本系统采用分层架构设计，遵循高内聚低耦合原则，通过角色权限管控实现管理员与普通用户的差异化操作；数据层面依托多表关联设计，冗余字段与索引优化提升查询效率，完整覆盖驾考学习、模拟考试、后台管理全场景需求，系统可维护性、扩展性强。
